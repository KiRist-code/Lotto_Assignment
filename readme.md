# C-Assignment

## Build

```bash
cmake -D CC=/usr/local/bin/gcc-13 -D CXX=/usr/local/bin/g++-13 . -B buildcmake -D CMAKE_C_COMPILER=/usr/local/bin/gcc-13 -D CMAKE_CXX_COMPILER=/usr/local/bin/g++-13 . -B build

cd build

make
```
